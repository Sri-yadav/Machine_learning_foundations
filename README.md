## Description

This project is based on the book [The Hundered Page Machine Learning Book](http://ema.cri-info.cm/wp-content/uploads/2019/07/2019BurkovTheHundred-pageMachineLearning.pdf). It summarizes the key concepts of Machine Learning through jupyter notebooks and codes written in python.

## Usage

This repository contains concepts and example code based on the book - "The Hundered Page Machine Learning Book"

You can use this project for:

- Reviewing key concepts of Machine Learning from the book in a summarized format.
- Understanding implementations of the book's ideas through simple Python code examples.
